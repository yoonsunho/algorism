# 1. 논리와 증명
12. n^2 이 3의 배수이면 n은 3의 배수임을 증명해라.
-> 대우 증명
n이 3의배수가 아니면 n^2도 3의 배수가 아니다.
n 이 3 의 배수 가 아니면 다음과  같이 표현할 수 있고
n = 3k + 1 -> n^2 = 9k^2 + 6k + 1 = 3(3k^2 + 2k) + 1
n = 3k + 2 -> n^2 = 9k^2 + 12k + 4 = 3(3k^2 + 4k + 1) + 1
n^2 도 3 의 배수가 아님을 확인할 수 있다

# 2. 기초 수식
4. T(n) = T(n/2) + 1, T(1) = 1
T(n) = T (n/2^k) + k
k= log n
T(n) = T(1) + k = 1 + log n
=> O(log n)
