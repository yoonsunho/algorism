# 3-1.
T(n) = T(n-1) + 1, T(0) = 1 의 식에서 n에 n-1을 대입해보면 T(n-1) = T(n-2) + 1이다.
따라서
T(n) = T(n-2) + 1 + 1 이다.
이와 같은 방법으로 n 에 n-3을 대입하여 식을 구해보면
T(n) = T(n-3) + 1 + 1 + 1 이다.
따라서 T(n) = T(n-k) + k로 정리할 수 있고, k = n을 대입하면
T(n) = T(0) + n = 1+n으로 정리된다
T(n) = 1+n 이므로 빅오 표기법으로 O(n+1)이고 이는 O(n)이라 표현한다.
-> O(n)

# 3-2.
명제: 자연수 n에 대해, n^2+5가 홀수이면 n은 짝수이다.
대우 : 자연수 n에 대해, n이 홀수미연 n^2 + 5가 짝수이다.
n 이 홀수이므로 n = 2k + 1(k 는 0 이상의 정수) 라 표현할 수 있다.
n^2 + 5 이 식에 대입해보면
(2k + 1)^2 + 5 =4k^2+4k+6 = 2(2k^2+2k+3)으로 2 로 나누어 떨어지는것을 확인할 수 있다.
2로 나누어 떨어지므로 짝수이다. 대우가 참 이므로
명제도 참이다.
-> 참
